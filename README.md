<h2 align="center">CRLFsuite</h2>

CRLFsuite is a fast tool specially degined to scan CRLF injectio. CRLFsuite supports single URL, multiple URLs, cookies, GET/POST methods.

### Installation

```
$ git clone https://github.com/Nefcore/CRLFsuite.git
$ cd CRLFsuite
$ sudo python3 setup.py install
$ crlfsuite -h
```
